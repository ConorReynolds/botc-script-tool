Unofficial Script Tool for Blood on the Clocktower

This is a tool for creating your own custom scripts for Blood on the Clocktower – similar to the official script tool, but with some improvements.

You can read the documentation on GitHub for a detailed feature list and usage guide, but here’s a quick summary of the basics:

  • to add characters, use the quick-add box on the top-right, or use the character-selector sidebar, which you can open by clicking the button on the middle-right of the screen
  • to remove characters, simply click on the character’s icon
  • to add another script, open the script-selector sidebar by clicking the button on the middle-left of the screen, then click the New Script button
  • to import a script JSON, use the button on the top right
  • to export a script, you have a number of options: printing , exporting the JSON , or creating a script link
  • to undo or redo the most recent change to the script, use the undo or redo button on the bottom right

You can read about the more advanced features – searching by ability text, view/lock mode, adding custom characters/rules/jinxes – in the documentation.

All your scripts are saved in your browser’s cache – so even if you close this tab, everything will be the way you left it.
